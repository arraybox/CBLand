1️⃣ 01_chungbuk_yearly_full_data.csv
   ├─ 내용: 2017-2025년 연도별 전체 데이터 (14개 지역)
   ├─ 행: 98개 (7개 연도 × 14개 지역)
   ├─ 컬럼: 연도, 행정구역명, 임야·농경지·대지·공장용지(면적+비율), 총면적
   └─ 용도: 시간 시계열 분석, 지역별 추이 파악

2️⃣ 02_chungbuk_yearly_summary.csv
   ├─ 내용: 연도별 전체 충청북도 합계 통계
   ├─ 행: 7개 (2017, 2018, 2019, 2020, 2023, 2024, 2025)
   ├─ 컬럼: 연도, 지역수, 토지용도별 면적+비율, 총면적
   └─ 용도: 도 전체 추세 분석, 도시화·산업화 진행도 파악

3️⃣ 03_chungbuk_change_rate_2017_2025.csv
   ├─ 내용: 2017년 vs 2025년 토지 용도별 변화율
   ├─ 행: 14개 (14개 지역)
   ├─ 컬럼: 지역명, 토지용도별(2017값, 2025값, 변화율%)
   └─ 용도: 지역별 개발 강도 비교, 급변 지역 식별

4️⃣ 04_chungbuk_cluster_analysis_2025.csv
   ├─ 내용: 2025년 기준 K-Means 군집분석 결과
   ├─ 행: 14개 (14개 지역)
   ├─ 컬럼: 지역명, 군집분류(도시/산업형·농업/산림형·균형형), 토지용도비율
   ├─ 군집 구분:
   │  ├─ 도시/산업형 (4개): 청주 흥덕·청원구, 진천군, 음성군
   │  ├─ 농업/산림형 (8개): 청주 상당구, 충주, 제천, 보은, 옥천, 영동, 괴산, 단양
   │  └─ 균형형 (2개): 청주 서원구, 증평군
   └─ 용도: 지역 유형 분류, 정책 차등 수립

5️⃣ 05_chungbuk_correlation_analysis.csv
   ├─ 내용: 토지 용도간 피어슨 상관계수 분석 (2025년)
   ├─ 행: 6개 (6개 용도 조합)
   ├─ 컬럼: 변수1, 변수2, 상관계수, p-value, 유의성
   ├─ 주요 결과:
   │  ├─ 임야 ↔ 농경지: 0.7114 ** (강한 양의 상관)
   │  ├─ 농경지 ↔ 대지: 0.5759 * (중간 양의 상관)
   │  └─ 대지 ↔ 공장용지: 0.5251 ns (약한 양의 상관)
   └─ 용도: 용도 간 의존성 파악

6️⃣ 06_chungbuk_regression_analysis.csv
   ├─ 내용: 선형 회귀분석 (대지면적 → 공장용지면적)
   ├─ 행: 1개 (회귀모델 1개)
   ├─ 결과: y = 0.6453×X - 1,884,492 (R² = 0.2758)
   ├─ 해석: 대지 1㎡ 증가 시 공장 약 0.65㎡ 증가
   └─ 용도: 변수 간 인과관계 분석

📊 분석 통계
───────────────────────────────────────────────────────────────
• 분석 기간: 2017-2025 (7개 연도)
• 대상 지역: 충청북도 14개 시군구
• 주요 토지 용도: 임야, 농경지(전+답), 대지, 공장용지
• 총 데이터 포인트: 98개 행 (7년 × 14개 지역)

📈 주요 발견
───────────────────────────────────────────────────────────────
✓ 임야·농경지 감소: -0.2% ~ -4.7%
✓ 대지·공장 증가: +7.8% ~ +21.8% (대지), +3.9% ~ +48% (공장)
✓ 공장용지 최대 증가 지역: 영동군 (47.95%)
✓ 도시/산업형 지역: 청주, 진천, 음성 (개발 활발)
✓ 농업/산림형 지역: 충주, 제천, 보은, 단양 (보전 우선)
