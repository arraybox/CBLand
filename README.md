# 충청북도 지적통계 토지 이용 현황 분석 데이터 패키지

본 프로젝트는 2017년부터 2025년까지 충청북도 14개 시군구의 지적통계 데이터를 기반으로 토지 이용 현황 및 변화 추이를 분석한 데이터 패키지입니다. Python을 활용하여 시계열 분석, 상관분석, 군집분석(K-Means) 등을 수행하였으며, 그 결과를 정량적 데이터와 시각화 자료로 제공합니다.

## 📁 데이터 파일 구성

본 패키지는 총 9개의 CSV 파일로 구성되어 있습니다.

### 1. 기본 데이터
* **01_chungbuk_yearly_full_data.csv**: 14개 시군구의 연도별 전체 데이터 (임야, 농경지, 대지, 공장용지 등 면적 및 비율).
* **02_chungbuk_yearly_summary.csv**: 충청북도 전체 합계 통계 데이터.
* **03_chungbuk_change_rate_2017_2025.csv**: 2017년 대비 2025년 토지 용도별 누적 변화율.

### 2. 분석 데이터
* **04_chungbuk_cluster_analysis_2025.csv**: 2025년 기준 K-Means 군집분석 결과 (도시/산업형, 농업/산림형, 균형형).
* **05_chungbuk_correlation_analysis.csv**: 토지 용도 간 피어슨 상관계수 분석 결과.
* **06_chungbuk_regression_analysis.csv**: 대지면적과 공장용지면적 간의 선형 회귀분석 결과.
* **07_chungbuk_yearly_change_rate.csv**: 연도별 토지 용도 변화율 (전년 대비).
* **08_chungbuk_yearly_use_trend.csv**: 도 전체의 연도별 토지 용도 추세 데이터.
* **09_chungbuk_extreme_change_areas.csv**: 연도별 및 용도별 최대 증가/감소 지역 데이터.

## 📊 주요 분석 결과

### 1. 토지 용도 변화 추세 (2017-2025)
* **도시화 및 산업화**: 전 지역에서 임야와 농경지 면적은 감소하고, 대지와 공장용지 면적은 증가하는 경향을 보입니다.
* **공장용지 급증**: 영동군(+47.95%), 청주 청원구(+34.05%), 음성군(+25.62%) 등에서 공장용지가 크게 증가하였습니다.
* **상관관계**: 대지면적과 공장용지면적 간에는 양의 상관관계(r=0.525)가 확인되었으며, 대지 1㎡ 증가 시 공장용지 약 0.65㎡가 증가하는 경향을 보입니다.

### 2. 지역 유형 분류 (Cluster Analysis)
지적통계 특성에 따라 14개 시군구를 3가지 유형으로 분류하였습니다.
* **도시/산업형**: 청주(흥덕, 청원), 진천, 음성 - 도시화 및 산업단지 개발이 활발한 지역.
* **농업/산림형**: 충주, 제천, 보은, 옥천, 영동, 괴산, 단양, 청주(상당) - 산림과 농업 보전이 주를 이루는 지역.
* **균형형**: 청주(서원), 증평 - 도시 기능과 농림 기능이 균형을 이루는 지역.
