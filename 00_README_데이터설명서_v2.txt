📋 기본 데이터 (01~03)
──────────────────────────────────────────────────────────────

1️⃣ 01_chungbuk_yearly_full_data.csv
   ├─ 내용: 2017-2025년 연도별 전체 데이터 (14개 지역)
   ├─ 행: 98개 (7개 연도 × 14개 지역)
   ├─ 컬럼: 연도, 행정구역명, 임야·농경지·대지·공장용지(면적+비율), 총면적
   └─ 용도: 시간 시계열 분석, 지역별 추이 파악, 기본 통계

2️⃣ 02_chungbuk_yearly_summary.csv
   ├─ 내용: 연도별 전체 충청북도 합계 통계
   ├─ 행: 7개 (2017, 2018, 2019, 2020, 2023, 2024, 2025)
   ├─ 컬럼: 연도, 지역수, 토지용도별 면적+비율, 총면적
   └─ 용도: 도 전체 추세 분석, 도시화·산업화 진행도 파악

3️⃣ 03_chungbuk_change_rate_2017_2025.csv
   ├─ 내용: 2017년 vs 2025년 토지 용도별 변화율 (8년간 누적)
   ├─ 행: 14개 (14개 지역)
   ├─ 컬럼: 지역명, 토지용도별(2017값, 2025값, 변화율%)
   └─ 용도: 지역별 장기 개발 강도 비교, 급변 지역 식별

📊 변화 분석 데이터 (04~09) ⭐ NEW
──────────────────────────────────────────────────────────────

4️⃣ 04_chungbuk_cluster_analysis_2025.csv
   ├─ 내용: 2025년 기준 K-Means 군집분석 결과
   ├─ 행: 14개 (14개 지역)
   ├─ 컬럼: 지역명, 군집분류, 토지용도비율
   ├─ 군집 구분:
   │  ├─ 도시/산업형 (4개): 청주 흥덕·청원, 진천, 음성
   │  ├─ 농업/산림형 (8개): 청주 상당, 충주, 제천, 보은, 옥천, 영동, 괴산, 단양
   │  └─ 균형형 (2개): 청주 서원, 증평
   └─ 용도: 지역 유형 분류, 정책 차등 수립

5️⃣ 05_chungbuk_correlation_analysis.csv
   ├─ 내용: 토지 용도간 피어슨 상관계수 분석 (2025년)
   ├─ 행: 6개 (6개 용도 조합)
   └─ 용도: 용도 간 의존성 파악

6️⃣ 06_chungbuk_regression_analysis.csv
   ├─ 내용: 선형 회귀분석 (대지면적 → 공장용지면적)
   ├─ 행: 1개 (회귀모델 1개)
   └─ 용도: 변수 간 인과관계 분석

7️⃣ 07_chungbuk_yearly_change_rate.csv ⭐ NEW
   ├─ 내용: 연도별 토지 용도 변화율 (전년도 대비)
   ├─ 행: 98개 (7개 연도 × 14개 지역)
   ├─ 컬럼: 연도, 지역명, 토지용도별 면적 및 전년도 대비 변화율(%)
   ├─ 특징: 기준연도(2017)는 변화율 미계산, 2018~2025는 전년도 대비 변화율
   └─ 용도: 연도별 단기 변화 추이 분석, 급변 시점 파악

8️⃣ 08_chungbuk_yearly_use_trend.csv ⭐ NEW
   ├─ 내용: 연도별 토지 용도 추세 (도 전체 합계 + 전년도 대비 변화)
   ├─ 행: 7개 (2017~2025 연도)
   ├─ 컬럼: 연도, 토지용도별(면적, 비율, 전년도 대비 변화율%)
   ├─ 특징: 도 전체 추세를 한눈에 파악 가능
   └─ 용도: 도시화 가속화 추세 시각화, 추이 분석

9️⃣ 09_chungbuk_extreme_change_areas.csv ⭐ NEW
   ├─ 내용: 연도별 용도 변화 극값 지역 (최대증가/최대감소)
   ├─ 행: 48개 (6개 년도 구간 × 4개 용도 × 2가지 극값)
   ├─ 컬럼: 기간, 용도, 변화유형(최대증가/최대감소), 지역명, 변화율, 변화전/후
   ├─ 특징: 각 연도별로 용도별 최대 변화 지역 추적
   └─ 용도: 정책 집중 필요 지역 식별, 개발 핫스팟 추적
